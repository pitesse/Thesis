\paragraph{Implemented change in \texttt{PitStopEvaluator}.}
The operator now includes two explicit pace-based recovery paths, \texttt{RIVAL\_PIT\_PACE\_SHIFT} and \texttt{SAFETY\_TIMER\_PACE\_SHIFT}, used only when direct post-pit rival gap reconstruction fails. Both paths require local green-lap evidence for driver and rival; otherwise, the output remains unresolved. This implementation keeps the streaming state machine deterministic and auditable while incorporating a lap-time-consistent fallback aligned with the race-time decomposition and optimization literature \citep{heilmeier2020race, carrasco2023optimization, quiroga2024optimization}.
