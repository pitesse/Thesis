## Note for Professors/assistants

Professors/assistants must use this template to create a new repository to be assigned to the student to carry out the project. This is the normal process that you will have to follow in order to interact with the template:

1. Go to the official repository of [CapyMOA](https://github.com/adaptive-machine-learning/CapyMOA);
2. Click the Fork button in the top right corner;
3. **KEY STEP**: On the page that opens ("Create a new fork"), look for the Owner dropdown menu.
    - Click the dropdown and select your `Streaming-Data-Analytics`.

4. Rename it replace **XXXX-XXXX** with the years of the course, the **CapyMOA** Prefix, the **Project-Title** with the project number and title i.e., `2025-2026_1-CapyMOA-SMLForSomething`;
5. Add a description;
6. Make it public;
7. Update this Readme file with the project description document;
8. Add the selected student as a collaborator.

Then add the standard `README.md` file to the repo.
